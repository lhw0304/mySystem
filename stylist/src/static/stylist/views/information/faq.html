How to embed a tagged photo into your blog?

1. Copy Code
In "Taggified Images" page, click embed the photo into yout blog button below the specific photo you want to embed.

In the popup, click Copy To Clipboard. Then you have successfully copied the code of the tagged photo to clipboard.
2. Paste Code Go to your blog and create or edit an existing post. Select the edit mode:   If you have Blogger: select the edit mode HTML.

If you have Wordpress.org: select the edit mode Text.
3.Success! Now your blog readers can see your tagged photo.
Views and generated through tagged photos on your blog will also be counted and earn commission for you.
